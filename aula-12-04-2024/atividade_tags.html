Tags de código

Apresentar blocos de código em documentos HTML tornou-se uma atividade bastante comum, visto que existem inúmeros sites que precisam exigir trechos de código. E não apenas código, também precisamos mostrar a saída do código ao tentar replicar janelas de terminais. A seguir vamos apresentar 3 elementos HTML utilizados para representar trechos de código e a saída do código com exemplos de cada uma dessas tags sendo utilizadas.

Code

O elemento HTML code apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de código. Por padrão, o conteúdo é exibido utilizando a fonte monoespaçada padrão do user agent's (en-US).

Para representar múltiplas linha de código, envolva a tag code dentro da tag pre. O elemento code por si só serve apenas para marcar uma única senteça ou linha como código.Uma regra CSS pode ser definida para o seletor code para sobrescrever a fonte padrão do navegador. Preferências definidas pelo usuário podem ter precedência sore o CSS especifciado.

Samp

A palavra "samp" pode ser traduzida como "amostra", uma parte de algo, nesse caso uma parte do código. O elemento HTML samp é utilizado pra englobar textos que representam partes de um código. Seu conteúdo é normalmente renderizado usando a fonte monoespaçada padrão do navegador (como o Courier ou o Lucida Console).

Pre

O elemento "Preformatted Text Element" ou texto pré-formatado é utilizado para inserir um conteúdo que se deseja seja visualizado do modo como foi editado. Para destacar o conteúdo a fonte visualizada é "monospace". Um texto dentro da tag pre é tipicamente exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo. Espaços em branco são mantidos no texto da mesma forma em que este foi digitado.

Kdb

O elemento HTML kdb representa um período de texto em linha que devolveu a entrada do usuário textual de um teclado, entrada de voz ou qualquer outro dispositivo de entrada de texto, ou seja uma tecla. Por convenção, o agente do usuário não faz o padrão para renderizar o conteúdo de um kbd element usando sua fonte padrão monospace, embora isso não seja exigido pelo padrão HTML.
